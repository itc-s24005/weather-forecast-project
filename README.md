# 天気予報アプリ
## 概要
検索した地域の天気情報を表示してくれます  
学校の先生のお力も借り仕上げました
## 使用技術
* Next.js（App Router)
* TypeScript
* Tailwind CSS
* React
## 使用API
[OpenWeatherMap API](https://openweathermap.org/api)
