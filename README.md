# 天気予報アプリ
## 概要
検索した世界の地域の天気情報(天気、気温、最高気温、最低気温、風速、湿度)を表示するサイトです  
学校の先生の力もお借りして仕上げました（一部ChatGPT使用）  
[**サイトはこちら**](https://weather-forecast-project-delta.vercel.app/)
## 使用技術
* Next.js（App Router)
* TypeScript
* Tailwind CSS
* React
## 使用API
[OpenWeatherMap API](https://openweathermap.org/api)  
指定された地域の天気情報を取得
